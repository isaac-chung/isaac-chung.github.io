# Classifying Real vs. Fake News Headlines
This exercise is taken from the University of Toronto Introduction to Machine Learning course (CSC2515) in Fall 2018
 assignment 1. A data set of 1298 'fake news' headlines and 1968 'real' news
 headlines. The data cleaning has been omitted, and the cleaned-up data are available as `clean_real.txt` and 
 `clean_fake.txt`. 

For reference, the real news headlines are taken from the [Million Headlines](https://www.kaggle.com/therohk/million-headlines) 
dataset, which contains news headlines published over a period of seventeen years (2003-2019) by ABC (Australian
Broadcasting Corp). The fake news headlines are taken from the [Fake News](https://www.kaggle.com/mrisdal/fake-news)
dataset, which contains text and metadata scraped from 244 websites. More details of the contents can be found in
the links.

## Dependencies
Make sure all of the following packages:
* pandas
* sklearn
* graphviz
* pydotplus
* matplotlib

## Usage
This assignment has four files:
* cleaned-up data are available as `clean_real.txt` and `clean_fake.txt`
* a [jupyter notebook](https://github.com/isaac-chung/CSC2515-coursework/blob/master/assign1/real_vs_fake_news.ipynb) 
containing functions and step-by-step guide for the assignment
* visualization of the decision tree `tree.png`
