# Seasonal Demand Forecasting
This mini-project serves as a familiarization exercise with time-series forecasting. Here we
study the historical product demand for a manufacturing company with footprints globally taken from 
[Kaggle](https://www.kaggle.com/felixzhao/productdemandforecasting).

The purpose of the project is to showcase how to approach a time-series exercise, including 
* data cleaning
* data decomposition
* data visualization
* make forecast predictions


## Dependencies
Make sure all dependencies are installed from 'requirements.txt'. 
```
pip install -r requirements.txt
```
